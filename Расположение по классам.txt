Главное окно (Tk)
    содержит классы (см. ниже)
    настраивает их
    организует пересылку данных
    Канвас (Canvas)
        получает настройки от главного окна
        содержит список prg
        показывает список prg
        перерисовывает по команде от главного окна
        выдает список фильтров
    Группа настроек (Frame)
        получает мастера
        выдает файл/каталог
        посылает изменение масштаба
        смещает опорную точку
        посылает показать/скрыть описание
    Группа вспомогательной информации (Frame)
        получает мастера
        текущий файл
        масштаб
    Группа фильтров
        получает мастера
        показывает список фильтров
        при выделении - выдает текущий
        при снятии - выдает сигнал отсутствия фильтра
    Настройки
        текущий каталог
        текущий файл
        список файлов
        опорная точка
        масштаб
